# Visual Grammar

Status: draft; detailed examples will be added with the M2 planner fixtures.

The planner should segment by semantic function rather than sentence boundaries. It must prefer the speaker for personal experience, emotion, transitions, conclusions, and calls to action. It should request stock media only when a visible object, place, action, process, or concrete example materially improves understanding.

Avoid generic visual metaphors for abstract words such as success, effort, future, technology, and growth. Search queries should describe observable subjects, actions, settings, composition, and shot types.

The planner may recommend title cards, structured graphics, screen captures, or user-owned assets, but Phase 1 only searches stock assets.
